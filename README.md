# Data-Analysis-using-Interactive-Dashboard
## Project Objective
Developed an interactive Excel dashboard to analyze 2023 online sales performance across months, states, product categories, customer demographics, platforms, and order status using slicers and dynamic charts.

## Dataset Used
- <a href="https://github.com/devrajwrk-coder/Data-Analysis-using-Interactive-Dashboard/blob/main/Online%20Sale%20Data%20Analysis.xlsx">Dataset</a>

## Questions (KPIs)
- Which month recorded the highest total sales in 2023?
- Which month had the lowest sales and what could be the reason?
- What is the total annual sales value across all months?
- Which state contributes the highest share of total sales?
- Which state has the lowest sales and requires improvement?
- Which online platform generates the maximum sales share?
- Which age group places the highest number of orders?
- What is the overall order fulfillment rate?
